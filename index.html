PokéGen — Browse Pokémon by Generation

PokéGen is a Vue 3 + Vite app that allows browsing Pokémon by generation, searching by name, and reading detailed information on stats, evolutions, and lore.

Main Features

  • Navigation by generation with an ordered list of Pokémon.
  • Pokémon detail page with complete card, stats, flavor text, height/capture rate, and evolution chain.
  • Search with dedicated input and debounce based on local PokeAPI index.
  • IndexedDB + InMemory cache with HTTP retry and jittered backoff.
  • Lazy loading images with Intersection Observer, skeleton, and SVG fallback.
  • SEO for route with OG/Twitter meta tags and JSON-LD.
  • Legal pages: Privacy Policy and Terms of Service.

Technical Stack

Frontend in Vue 3 with Vite, Vue Router, and Pinia. HTTP Client Axios with retry and cache. Styling with Tailwind 4. Testing with Vitest.

Architecture

Clean Architecture with core, infrastructure, app, shared, and modules (PokéGen) layers. Dependency Injection through AppContainer and feature containers.
